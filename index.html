PROJECTS

Over the course of my studies and hands-on experiences, I’ve worked on various projects that reflect my passion for web development, design, and system management. Each project has allowed me to deepen my technical knowledge while solving real-world challenges. Below are some of the key projects I’ve completed:

Guidance Counseling: Student Information Sheet (SIS) & Student Information Updating Form (SIUF) System.

Developed an online system to manage student records and update information for the university’s Guidance Counseling department. This system helps counselors track student progress and maintain up-to-date records, ensuring accurate and efficient case management. Built using C++ for backend logic, along with HTML, CSS, and JavaScript for the front-end to create a user-friendly interface, and SQL for securely storing and managing student data.

BSU: Course Management System


A comprehensive platform for managing online learning, assignments, and student assessments, built using technologies such as HTML, CSS, JavaScript, and JSON for the front-end, along with Java and MySQL for the back-end. This system enhances the educational experience by providing easy access to course materials and performance tracking, utilizing efficient data management and seamless integration between the front and back ends.

Code Red: Revolutionizing Learning with Productivity Fusion in LMS

Developed as a comprehensive platform for managing appointments, the Code Red system addresses challenges like inefficient consultation scheduling and limited integration functionalities, ultimately enhancing the educational experience at BSU. Built using HTML, PHP, JavaScript, and CSS, it provides a robust and user-friendly interface, facilitating better access to information and resources for both lecturers and students.
